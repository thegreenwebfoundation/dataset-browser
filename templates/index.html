Welcome to the Green Web Foundation dataset browser.

Below is a browseable daily snapshot all the green domains tracked in the Green Web Foundation database, along with who they are hosted by, and when they were last updated.

This data is updated every night at around midnight, and the tables are sortable, and searchable. If you know SQL, can run queries agains the data in-browser, and download the data for further analysis or use as as JSON, or in CSV format.

See the latest green domains snapshot, or learn more about the data.
