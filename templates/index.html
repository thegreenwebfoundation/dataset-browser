Welcome to the Green Web Foundation dataset browser.

Green domains dataset

Below is a browseable daily snapshot of all the green domains tracked in the Green Web Foundation database from the last twelve months, along with who they are hosted by, and when they were last updated.

This data is updated every night at around midnight, and the tables are sortable, and searchable. If you know SQL, can run queries agains the data in-browser, and download the data for further analysis or use as as JSON, or in CSV format.

See the latest green domains snapshot, or learn more about the data.

Real time cloud dataset

Since 2023, The Green Web Foundation has been a member of the Green Software Foundation Real Time Cloud working group, a project that has set out to define consistent, standard ways to report the carbon intensity of the computing hours sold by large cloud providers.

We have republished the key dataset created by the working group, the Real Time Cloud Metadata dataset, here on this site, in a form where it can be easily queried, linked to, and downloaded.

See this link to documentation on the main project website to understand what each column in the dataset means, and how it was created.

See the latest (v1.0) snapshot of the real time cloud metadata dataset
